## 🧬 1. DNA יומי - ניתוח אישיות יומית

**קושי: ⭐⭐⭐**

רעיון חדשני לגמרי! האפליקציה תייצר כל יום "DNA יומי" - ויזואליזציה ייחודית שמייצגת את היום שלך:

- **Helix גנטי מותאם אישית** - כל פעולה שלך משנה את צורת ה-DNA
- **צבעים דינמיים** - ספורט = כחול, עבודה = כתום, מנוחה = ירוק
- **השוואת DNA** - ראה איך ה-DNA של היום משתווה לימים קודמים
- **"DNA אידיאלי"** - הגדר DNA יעד ונסה להגיע אליו

```
מבנה נתונים:
{
  dailyDna: {
    date: string,
    segments: DnaSegment[],
    overallPattern: 'productive' | 'balanced' | 'creative' | 'restful',
    visualHash: string, // unique daily pattern
    similarity: number // compared to ideal DNA
  }
}
```
